## Spotify Profile Viewer (Full Stack)

A full-stack project that uses the Spotify Web API to authenticate users and fetch profile data.

### Structure
- `React stuff/` – React (Vite) frontend
- `Backend/` – Node.js + Express backend for Spotify OAuth

### Features
- Spotify OAuth authentication
- Fetch and display user profile data
- Clean API integration

### Planned
- AI-based music taste roasting (Gemini API)
